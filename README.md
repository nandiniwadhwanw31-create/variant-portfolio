# Variant Portfolio

Retro-futuristic “OS” portfolio for a Creative Technologist — React + Vite + Tailwind.

## Run locally

```bash
npm install
npm run dev
```

Open the URL shown in the terminal (typically http://localhost:5173).

## Custom images

Replace files in `public/images/` to use your own photos (keep the same filenames):

| File | Used for |
|------|----------|
| `profile.jpg` | Profile panel (circular + background) |
| `project-01.jpg` … `project-06.jpg` | Archive project card previews |

Edit copy in `src/data/site.ts` and project details in `src/data/projects.ts`.

## Features

- Windowed hero, bio, and six archive project cards
- Pink / lavender vaporwave palette with CRT scanline overlay
- Ghost Protocols full-screen overlay (`#ghost-overlay` hash target)
- Fragment Archive map with SVG connection lines
- Smooth scroll navigation and “Initialize Project Grid” CTA
